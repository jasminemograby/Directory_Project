# איך למצוא Connection String ב-Supabase

## מיקומים אפשריים:

### אפשרות 1: Settings → Database → Connection string
1. לך ל-**Settings** (ההילינגים בצד)
2. לחץ על **Database**
3. גלול למטה - חפש:
   - **Connection string** section
   - או **Connection info**
   - או **Database connection**

### אפשרות 2: Settings → API
1. לך ל-**Settings** → **API**
2. חפש:
   - **Database URL**
   - או **Connection string**
   - או **PostgreSQL connection**

### אפשרות 3: Project Settings → Database
1. בדף הראשי של הפרויקט
2. חפש **Database** או **Connection**
3. לחץ על זה

### אפשרות 4: SQL Editor
1. לחץ על **SQL Editor** בתפריט הצד
2. לפעמים יש שם קישור ל-Connection info

---

## מה לחפש:

חפש אחד מהבאים:
- **Connection string**
- **Database URL**
- **PostgreSQL connection**
- **Connection info**
- **Database connection**
- **Connection pooling**
- **URI**
